# 株式会社山村書店 公式サイト

岐阜県安八郡神戸町にある書店・教科書取扱店のコーポレートサイトです。

**公開URL:** https://www.yamamurabook.shop/  
**GitHub Pages:** https://sakasaew.github.io/yamamurabooks/

---

## ページ一覧

| ファイル | ページ名 |
|---|---|
| index.html | トップページ |
| access.html | 利用案内（店舗・倉庫情報） |
| company.html | 会社概要 |
| recruit.html | 採用情報 |
| faq.html | よくある質問 |
| contact.html | お問い合わせ |
| sakasaew.html | 長女の部屋 |
| privacypolicy.html | プライバシーポリシー |

## 技術スタック

- 静的HTML（フレームワーク不使用）
- [Tailwind CSS](https://tailwindcss.com/) CDN（インラインconfig）
- [M PLUS 1p](https://fonts.google.com/specimen/M+PLUS+1p)（Google Fonts）
- お問い合わせフォーム：[Formspree](https://formspree.io/)
